## ١. مقدمة المشروع والرؤية الاستراتيجية

مشروع **درع الفطرة (FitraShield)** هو منظومة حماية رقمية متكاملة تهدف إلى صيانة الفطرة السليمة وحماية براءة الأطفال وعقول الشباب من مخاطر المحتوى الإباحي على شبكة الإنترنت. يعمل النظام بالكامل على جهاز المستخدم دون الحاجة إلى سيرفرات خارجية أو تسجيل دخول أونلاين، مما يضمن **الخصوصية المطلقة والسرعة الفائقة**.

### ١.١ الهوية الرسمية للمشروع

| البند | التفصيل |
|---|---|
| الاسم بالعربية | درع الفطرة |
| الاسم بالإنجليزية | FitraShield |
| نوع الأداة | إضافة متصفح (Chrome Extension) — Manifest V3 |
| المتصفحات المدعومة | Google Chrome / Microsoft Edge / Brave / Opera (نواة Chromium) |
| نظام التشغيل المستهدف | Microsoft Windows (مع دعم مستقبلي لـ macOS وLinux) |
| نموذج العمل | مجاني بالكامل — قائم على الدعم والتبرعات (Donation-based / Open-Source) |
| الجمهور المستهدف | الآباء والأمهات في الوطن العربي (مصر، الخليج، شمال أفريقيا) |

### ١.٢ المبادئ الأساسية للمشروع

- **الخصوصية المطلقة:** العمل محلياً 100% — لا بيانات تغادر الجهاز
- **الحرية في الاستخدام:** مجاني بكافة المميزات لجميع الفئات بلا استثناء
- **الأمان الثنائي:** رقابة صارمة مع الحفاظ على كفاءة الإنترنت الطبيعي
- **التوسعية:** معمارية قابلة للتحول مستقبلاً لمنصة SaaS متكاملة
- **الهوية العربية:** تركيز متخصص على المحتوى الإباحي الموجه للشرق الأوسط وشمال أفريقيا

### ١.٣ السياق والدوافع

المواقع الإباحية الموجهة للشرق الأوسط (مصر، الخليج، شمال أفريقيا) تتنامى بسرعة وتغير نطاقاتها (Domains) باستمرار للتحايل على الأنظمة الحالية. فضلاً عن ذلك، المحتوى المكتوب بالعامية العربية نادراً ما يُرصد في قوائم الحجب الأجنبية. تأتي أداة درع الفطرة لتملأ هذا الفراغ بأداة **عربية الهوية، مجانية، وصعبة الالتفاف عليها**.

> 💡 **الهدف الجوهري:** حماية عقول وفطرة أطفالنا من المحتوى الإباحي كرسالة اجتماعية ودينية راسخة

## ٢. المعمارية الهندسية للنظام (System Architecture)

يعتمد نظام FitraShield على تكامل وثيق بين **طبقتين حمائيتين**: طبقة المتصفح (Browser Level) وطبقة نظام التشغيل (OS Level)، لمنع أي محاولة للالتفاف أو الحذف أو التعطيل.

### ٢.١ مخطط المعمارية الكلية

```
[ متصفح Chromium ]  ◄──►  [ إضافة FitraShield (MV3) ]
                                         |
                              (Native Messaging API)
                                         ▼
[ Windows OS ]  ◄──►  [ FitraShield_Helper.exe (Admin) ]  +  [ Registry HKLM ]
```

### ٢.٢ ملفات الإضافة داخل المتصفح (Extension Components)

| الملف | الوظيفة |
|---|---|
| `manifest.json` | الملف التعريفي الأساسي — يمنح الأداة هويتها وصلاحياتها من جوجل (MV3) |
| `background.js` (Service Worker) | العقل المدبر — يراقب حركة الشبكة ويطبق الحجب الفوري بتقنية Keep-Alive |
| `popup.html / popup.js` | النافذة السريعة عند الضغط على الأيقونة — تطلب كلمة المرور |
| `options.html / options.js` | لوحة التحكم الكاملة للآباء — إدارة القوائم والتقارير والإعدادات |
| `blocked.html` | صفحة الحجب التربوية — تظهر للطفل عند محاولة الدخول لموقع محظور |
| `rules.json` | ملف قواعد الحجب المدمجة — آلاف الدومينات جاهزة ومحملة محلياً |
| `keywords.json` | قاعدة الكلمات المفتاحية بالعربية الفصحى والعامية واللهجات الإقليمية |

### ٢.٣ المكونات المساعدة على مستوى نظام التشغيل (OS Components)

| المكوّن | الوظيفة |
|---|---|
| `FitraShield_Enforcer.reg` | ملف ريجستري لويندوز — يطبق سياسات المؤسسات لقفل المتصفح ومنع حذف الإضافة |
| `FitraShield_Helper.exe` | برنامج صغير بصلاحيات Admin — يحفظ التقارير على الهارد ويعيد تعيين الباسورد |

### ٢.٤ الصلاحيات المطلوبة في `manifest.json`

- `declarativeNetRequest` — حجب الشبكة من المنبع قبل تحميل أي محتوى
- `storage` — التخزين المحلي المشفر لكلمة المرور والقوائم والتقارير
- `tabs` — مراقبة التبويبات وتوجيه الحجب
- `nativeMessaging` — التواصل مع برنامج المساعد الخارجي لحفظ التقارير
- `webNavigation` — رصد محاولات التنقل قبل التنفيذ

## ٣. محرك الفلترة والحجب الذكي (Smart Filtering Engine)

تعتمد آلية الحجب على **ثلاث ركائز متكاملة** تضمن عدم إفلات أي موقع إباحي حتى لو تغير نطاقه أو استخدم دومين جديد.

### ٣.١ الركيزة الأولى — قاعدة البيانات المدمجة (Pre-loaded Blocklist)

ملف `rules.json` مدمج داخل الأداة يحتوي على آلاف الدومينات المحجوبة مسبقاً من مصادر موثوقة. يتم تحميلها بسرعة البرق دون استهلاك ملحوظ للرامات عبر الـ `declarativeNetRequest API`.

- **المواقع الإباحية العالمية:** مستقاة من قوائم مجتمعية محدثة (StevenBlack، OISD، Shalla's Blacklists)
- **المواقع الموجهة للشرق الأوسط:** رصد متخصص للمواقع الناطقة بالعربية والمستهدفة لمصر والخليج وشمال أفريقيا
- **قائمة أدوات الالتفاف:** مواقع تحميل الـ VPN والبروكسي الشهيرة مدرجة ضمن قائمة الحجب

### ٣.٢ الركيزة الثانية — محرك الكلمات المفتاحية (Keyword Inspection Engine)

المواقع الإباحية تغير دوماتها باستمرار. لذلك تفحص الأداة نصوص العناوين (Page Titles) والروابط (URLs) محلياً **قبل** تحميل أي شيء.

- الكلمات الجنسية الصريحة باللغة العربية الفصحى
- العامية المصرية والخليجية والمغاربية
- الكلمات المستعارة من اللغات الأجنبية المستخدمة محلياً
- عند رصد كلمة ممنوعة: الحجب الفوري + إضافة الدومين تلقائياً لقائمة الحجب الديناميكية

> 💡 هذا المحرك يضمن صد المواقع الجديدة حتى لو لم تُدرج بعد في القائمة المدمجة

### ٣.٣ الركيزة الثالثة — التحديث الصامت المجتمعي (Silent Community Updates)

- تعمل الأداة على تحميل ملف JSON محدث دورياً من مستودع GitHub مجاني ومفتوح
- التحديث يتم في الخلفية دون تدخل من الأب أو إشعار مزعج
- يُدمج الملف المحدث مع القائمة المحلية تلقائياً لتوسيع نطاق الحجب
- الآباء يستطيعون المساهمة بإرسال مواقع جديدة يكتشفونها

### ٣.٤ إجبار وضع البحث الآمن (SafeSearch Enforcement)

- إجبار Google على SafeSearch في جميع طلبات البحث
- إجبار YouTube على Restricted Mode لمقاطع الفيديو
- إجبار Bing وDuckDuckGo وباقي محركات البحث على نفس القيود

## ٤. منظومة سد الثغرات والتحصين الحديدي (Anti-Bypass Matrix)

| الثغرة المحتملة | حيلة الطفل للتخطي | خط الدفاع في درع الفطرة | الآلية التقنية |
|---|---|---|---|
| حذف الإضافة | الضغط على Remove من صفحة الإضافات | إخفاء وقفل زر الحذف تماماً | `ExtensionInstallForcelist` في الريجستري |
| تعطيل الإضافة | تبديل مفتاح التفعيل للإيقاف | قفل مفتاح التعطيل — غير قابل للضغط | Enterprise Policy عبر الريجستري |
| الإنهاء من Task Manager | `Shift+Esc` ثم `End Process` | إعادة التشغيل الذاتي الفوري | Keep-Alive في `background.js` |
| التصفح الخفي (Incognito) | فتح نافذة Incognito للتصفح بحرية | تفعيل الأداة إجبارياً في الوضع الخفي | سطر إجباري في ملف الريجستري |
| إنشاء بروفايل جديد | `Create New Profile` في المتصفح | حماية كل البروفايلات الحالية والمستقبلية | التثبيت في `HKEY_LOCAL_MACHINE` |
| تعطيل JavaScript | قفل الـ JS للموقع لتعطيل كود الحجب | الحجب من منبع الشبكة قبل تحميل أي كود | `declarativeNetRequest` |
| استخدام الـ VPN | تشغيل برنامج VPN لتغيير الـ IP | الاعتراض المحلي المسبق + حجب مواقع تحميله | الأداة تلتقط الرابط قبل تشفيره |
| إضافات VPN داخل كروم | تحميل إضافة VPN من Chrome Web Store | قفل Chrome Web Store نفسه بالقائمة السوداء | Custom Blacklist يحجب `chromewebstore.google.com` |
| متصفحات بديلة | تحميل Firefox Portable على USB | رصد أي متصفح غير خاضع للرقابة وإغلاقه | `FitraShield_Helper.exe` يراقب العمليات |

> ⚠ **تنبيه:** أي طفل يحاول تخطي الأداة سيواجه عائقاً في كل مسار، وستُسجل كل محاولة في ملف التقرير المحلي

## ٥. إدارة القوائم والبيانات (Data Management)

الإنترنت يظل مفتوحاً وطبيعياً 100% خارج نطاق القوائم المحظورة.

### ٥.١ آلية الحجب والاستثناء

- **القائمة المدمجة (Embedded Blocklist):** آلاف المواقع المحظورة مسبقاً داخل الأداة
- **القوائم المخصصة المبوّبة (Categorized Custom Lists):** الأب ينشئ قوائم ويسميها براحته
  - مثال: قائمة "ألعاب"، قائمة "تشتيت"، "مواقع التواصل وقت الامتحانات"
- **قائمة الاستثناءات (Exceptions List):** لتفادي الحجب الخاطئ للمواقع التعليمية والطبية

> 💡 **ملاحظة:** لا توجد قائمة بيضاء مغلقة — الإنترنت مفتوح بالكامل باستثناء المحظورات فقط

### ٥.٢ الاستيراد الجماعي الذكي (Smart Bulk Import)

يتيح للأب رفع ملف نصي (`.txt`) يحتوي على مئات الروابط دفعة واحدة، وتقوم الأداة بـ:

- حذف الروابط المكررة تلقائياً (Auto Deduplication)
- تنظيف الروابط الطويلة والاحتفاظ بالنطاق الأساسي فقط (URL Normalization)
  - مثال: `https://site.com/video/123?cat=adult` ← `site.com`
- تجاهل السطور الفارغة والأحرف الغريبة تلقائياً

### ٥.٣ التصدير ومشاركة القوائم مجتمعياً (Export & Share)

- تصدير القائمة السوداء كملف نصي (`.txt`) لمشاركتها مع آباء آخرين
- أي أب آخر يستطيع استيراد الملف بضغطة زر واحدة دون كتابة يدوية
- يُعزز التعاون المجتمعي في حماية الأولاد

### ٥.٤ النسخ الاحتياطي الكامل (Full Backup & Restore)

ملف JSON مشفر محلياً يحتوي على:

- كلمة المرور الرئيسية (مشفرة)
- القوائم السوداء المخصصة بأسمائها وتصنيفاتها
- قائمة الاستثناءات
- جميع إعدادات الأداة

> ✔ **عند إعادة تثبيت الويندوز:** يرفع الأب ملف الاستعادة — تعود جميع الإعدادات في أقل من ثانية

## ٦. نظام التقارير المحلي المتقدم (Local Reporting System)

### ٦.١ آلية حفظ التقارير

المتصفح لا يملك صلاحية حفظ ملفات مباشرة على الهارد. لذلك تُرسل الأداة بيانات الحجب عبر **Native Messaging** إلى `FitraShield_Helper.exe`.

- الأب يختار مسار الحفظ بحرية (أي بارتيشن أو فولدر)
- التقرير بصيغة **CSV** يُفتح بسهولة عبر Microsoft Excel
- **خاصية التمويه:** حفظ التقرير باسم مخفي (مثل `system_config.dat`) لحمايته

### ٦.٢ هيكل التقرير المنسق

```
══════════════════════════════════════════════════════════
[ملخص درع الفطرة] | محاولات اليوم: 3 | وقت الذروة: 09:30م | الحصن: نشط ✔
══════════════════════════════════════════════════════════
```

| الوقت والتاريخ 📅 | الموقع / البحث 🌐 | سبب الحجب 🛡 | التصرف الفوري 🛠 |
|---|---|---|---|
| اليوم — 09:30 مساءً | بحث جوجل عن: (مصطلح ممنوع) | كلمة ممنوعة في البحث | إغلاق الصفحة فوراً 🚫 |
| اليوم — 08:15 مساءً | `bad-website.xyz` | مدرج في القائمة المدمجة | منع الدخول وتحويل المسار 🚫 |
| أمس — 04:00 عصراً | `gaming-forum.net` | قائمة الأب المخصصة | منع الدخول وتحويل المسار 🚫 |

### ٦.٣ حقول بيانات التقرير

| الحقل | الوصف |
|---|---|
| التاريخ والوقت | يُسجل بدقة مع الثواني لمعرفة وقت الذروة الدقيق |
| اسم الموقع | النطاق الأساسي مختصراً (مع إمكانية رؤية الرابط كاملاً بالضغط) |
| سبب الحجب | هل هو من القائمة المدمجة أم المخصصة أم كلمة مفتاحية |
| الكلمة المستخدمة | تُسجل الكلمة التي أدت للحجب عند استخدام محرك البحث |
| التصرف المتخذ | وصف الإجراء الفوري: إغلاق / تحويل / منع |
| حالة الأداة | تأكيد أن الدرع كان نشطاً وفاعلاً لحظة الرصد |

## ٨. تحصين المتصفح عبر ملف الريجستري (Registry Enforcement)

هذا هو السر الذي تستخدمه الشركات الكبيرة والمدارس للتحكم الكامل في المتصفحات.

### ٨.١ ما الذي يفعله ملف الريجستري؟

- تثبيت الأداة وتفعيلها تلقائياً على **أي حساب** في الجهاز
- إخفاء وتعطيل زر الحذف — يظهر: `Managed by your organization`
- منع إيقاف تشغيل الأداة من صفحة الإضافات
- التطبيق على مستوى `HKEY_LOCAL_MACHINE` — ليس المستخدم الحالي فقط
- فرض تفعيل الأداة في وضع التصفح الخفي إجبارياً
- حماية كل البروفايلات الحالية والجديدة

### ٨.٢ كيف يُشغّل الأب ملف الريجستري؟

1. تُوفر الأداة ملف (`.reg`) للتحميل من لوحة التحكم
2. الأب يضغط عليه نقراً مزدوجاً ويوافق على التأكيد
3. في أقل من ثانية: الحماية تُفعَّل على مستوى الجهاز والمتصفح بالكامل

> ⚠ **تنبيه:** يجب تشغيل الملف من حساب مسؤول (Admin Account) — وحساب الطفل يجب أن يكون Standard User

## ٩. إدارة واسترداد كلمة المرور محلياً (Password Management)

### ٩.١ إنشاء كلمة المرور لأول مرة

- صفحة إعداد أولية تُعرض عند أول تثبيت
- الأب يختار كلمة مرور قوية
- يُولّد النظام تلقائياً **مفتاح استرداد رئيسي** — مثال: `FS-1092-ZX99-KA77-QP44`
- تُعرض رسالة: احفظ هذا المفتاح خارج الجهاز

### ٩.٢ مسارا الاسترداد

**المسار الأول — مفتاح الاسترداد:**
- الأب يضغط على "نسيت كلمة المرور" ويُدخل المفتاح الطويل
- النظام يُصفّر الباسورد ويسمح بضبط واحد جديد

**المسار الثاني — صلاحيات نظام التشغيل (Admin Override):**
- الأب يفتح `FitraShield_Helper.exe` بصلاحيات Administrator
- يضغط "إعادة تعيين كلمة مرور الأداة"
- البرنامج يمسح الباسورد القديم محلياً

> ⚠ يجب أن يكون حساب الطفل Standard User — هذا ما يمنعه من استخدام هذا المسار

## ١٠. التكامل مع حماية شبكة الواي فاي (Home DNS Integration)

### ١٠.١ كيف يعمل DNS العائلي؟

عندما يكتب أي جهاز في البيت اسم موقع، يذهب للراوتر الذي يسأل "دفتر عناوين الإنترنت" (DNS) عن رقم IP الموقع. **DNS العائلي هو دفتر عناوين نظيف** — المواقع الإباحية محذوفة منه كلياً، فلا تُفتح على أي جهاز.

### ١٠.٢ خدمات DNS العائلية الموصى بها

| الخدمة | DNS الأساسي | DNS الثانوي | المميزات |
|---|---|---|---|
| Cloudflare Family | `1.1.1.3` | `1.0.0.3` | سريع جداً — يحجب الإباحي والبرامج الضارة |
| AdGuard DNS Family | `94.140.14.15` | `94.140.15.16` | يحجب الإباحي + الإعلانات + التتبع |
| CleanBrowsing Family | `185.228.168.168` | `185.228.169.168` | متخصص في الفلترة العائلية |

### ١٠.٣ التكامل بين الطبقتين بدون تعارض

**الأداة لا تُعدّل إعدادات الشبكة أو الـ DNS في الكمبيوتر** — وهذا تصميم مقصود:

- **الراوتر + DNS العائلي:** يحمي الموبايلات والشاشات من المواقع المعروفة
- **درع الفطرة على الكمبيوتر:** يصد المواقع العربية الجديدة التي لم يرصدها DNS بعد
- **التقارير:** الـ DNS الراوتر لا يُنتج سجلاً — أما الأداة فتحفظ كل محاولة بالتفصيل

> ✔ طبقتان تحميتان متكاملتان: الراوتر يحمي البيت كله، والأداة تحمي الكمبيوتر بعمق وذكاء

## ١٣. ملاحظات ختامية للفريق التقني

### ١٣.١ أولويات التطوير

1. البدء بـ `manifest.json` وـ `background.js` مع `rules.json` أساسية للحجب التجريبي
2. اختبار تكامل الريجستري على بيئة Windows 10 و Windows 11
3. بناء واجهة `options.html` كأول لوحة تحكم للآباء
4. بناء صفحة `blocked.html` بالتصميم التربوي

### ١٣.٢ اعتبارات الأمان

> ⚠ يجب تشفير كلمة المرور محلياً باستخدام `bcrypt` أو `Argon2` — لا تخزينها كنص صريح

- يجب تشفير ملف النسخ الاحتياطي (`.json`) بكلمة مرور قبل الحفظ
- مفتاح الاسترداد يجب توليده بشكل عشوائي حقيقي (`crypto.getRandomValues`)
- التقارير يجب أن تُشفَّر في الملف المحفوظ لمنع قراءتها بدون الأداة

### ١٣.٣ مبادئ تجربة المستخدم (UX Principles)

- اللغة العربية الفصحى السهلة في جميع شاشات الأداة
- صفر مصطلحات تقنية أمام الآباء — توضيح كل شيء بلغة بسيطة
- عملية التثبيت في أقل من 3 دقائق للأب غير التقني
- التصميم المريح للعين: ألوان هادئة — أزرق داكن وأخضر ورمادي فاتح
